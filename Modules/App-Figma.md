# Figma Enhanced

Use Alt+HJKL to navigate, switch focus, zoom in and out in figma.
使用 Alt+HJKL 在 figma 中导航，切換焦点，縮放。

## 常用功能/特性

## 说明

| 模式          | Anki 增强模块 | 说明       |
| ------------- | :-----------: | ---------- |
| 在 figma 界面 |      `k`      | ↑ 切換元素 |
| 在 figma 界面 |      `j`      | ↓ 切換元素 |
| 在 figma 界面 |      `h`      | ← 切換層級 |
| 在 figma 界面 |      `l`      | → 切換層級 |
